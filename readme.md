
Apex Final Extreme
==================

Ce dossier contient une version finale statique du site avec :
- pages FR et EN séparées
- bouton de langue en haut
- section avant / après mise à jour avec de vraies paires d’images du même lieu
- images stockées localement dans assets/images/results

Fichiers principaux
-------------------
- index.html / index-en.html
- services.html / services-en.html
- contact.html / contact-en.html
- assets/css/style.css
- assets/js/main.js

Avant / après

Intégration WordPress
---------------------
Si vous voulez intégrer ces pages dans WordPress :
1. Mettez le CSS dans un snippet global ou dans votre thème enfant
2. Mettez le JS dans le footer
3. Collez le contenu de chaque page dans des pages WordPress
4. Uploadez les images du dossier assets/images/results dans la médiathèque si vous ne gardez pas les chemins locaux
